## **2. 프로젝트 요약**
-------------------------------------------------------------
### **2.1 데이터 전처리**

- 시각화 : Matplotlib, POWER BI, Tableau  
- 전처리 : pandas, numpy 

- 전처리 과정 : 외부데이터 크롤링 -> 시각화 분석 -> 이상치 보정 -> 유효한 데이터 선정 -> 파생변수생성 -> 최종데이터셋 완성


<br/>

### **2.2 모델 검증 및 성능 향상**
